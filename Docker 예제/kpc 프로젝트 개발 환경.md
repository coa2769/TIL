### fcs 개발 환경

- docker로 현재 pc에서 linux로 된 개발 환경을 만든다.
  - volume 으로 pc와 linux 개발 환경이 폴더를 바인딩한다.

- docker-compose 는 여러 docker container를 한 틀에서 관리해주는 툴이다.(쿠버네티스 대신 사용)
